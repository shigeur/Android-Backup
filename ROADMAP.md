# Roadmap

Android Backup is actively developed. While the core ADB transfer functionality and dual-pane file management are stable, there are many features planned for future releases.

## Upcoming Features

### Core Functionality
- [ ] **Incremental Backup:** Only copy files that have changed or been added since the last backup.
- [ ] **Restore Wizard:** A guided flow to restore backups back to the Android device.
- [ ] **Folder Synchronization:** Bi-directional or uni-directional automated syncing between folders.

### Workflow & Productivity
- [ ] **Backup Profiles:** Save presets for different devices or different backup scenarios.
- [ ] **Scheduled Backups:** Run backups automatically when the device is connected or at a specific time.
- [ ] **Search:** Quick fuzzy search for files within the file manager.

### Enhancements
- [ ] **Wi-Fi Debugging Support:** Seamlessly connect and transfer files over Wi-Fi without needing a USB cable.
- [ ] **Quick Look Integration:** Preview images and text files directly from the Android device before copying.

*Note: This roadmap is subject to change based on community feedback and developer availability.*
